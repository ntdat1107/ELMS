# ELMS
E-learning management system

# Option 1: npm
First, from the root, run the command:
```
cd .\React\my-app\
npx create-react-app <your-app-name>
```
and then run
```
npm install
```
to install all needed packages.
Then type `npm start` to run the UI of the website
