# ELMS Guildline  to  run  application  on  localhost
E-learning management system

# Option 1: npm
First, from the root, run the command:
```
cd .\React\my-app\
```
and then run
```
npm install
```
to install all needed packages.
To run the UI of the website, type the command
```
npm start
```
# Option 2: yarn
First, from the root, run the command:
```
cd .\React\my-app\
```
and then run
```
yarn install
```
to install all needed packages.
To run the UI of the website, type the command
```
yarn start
```
